# Industrial Training Kit Website

## Project Overview
This is a responsive multi-page website built as part of an Industrial Training internship project. It demonstrates the use of HTML, CSS, and JavaScript to create a simple educational training platform.

## Pages Included
- Home Page
- About Page
- Services Page
- Contact Page

## Features
- Responsive design for mobile and desktop
- Navigation menu across all pages
- Services displayed using card layout
- Contact form with basic JavaScript validation
- Clean and simple UI design

## Technologies Used
- HTML5
- CSS3 (Flexbox & Media Queries)
- JavaScript (Basic DOM manipulation)

## How to Run the Project
1. Download or clone the project folder
2. Open `index.html` in any web browser
3. Navigate through the pages using the menu
